# Monthly Stock Performance Analysis of NVIDIA (NVDA) Using WRDS CRSP Data

## Project Overview
This project analyses NVIDIA’s monthly stock performance since 2022 using CRSP data from WRDS. It focuses on stock price trends, monthly returns, and volatility.

## Analytical Problem
How has NVIDIA performed since 2022 in terms of stock price movement, monthly returns, and volatility?

## Target Audience
Retail investors and beginner financial analysts.

## Data Source
WRDS - CRSP Monthly Stock File (`crsp.msf`) and CRSP Monthly Stock Header File (`crsp.msfhdr`)  
Accessed on: 26 April 2026

## Tools Used
- Python
- pandas
- matplotlib
- seaborn
- WRDS / SQL

## Repository Contents
- `monthly_stock_performance_nvda_wrds.ipynb` - main notebook
- `README.md` - project description
- `reflection_report.pdf` - reflection report
- `images/` - exported charts

## Key Findings
- NVIDIA's stock price generally increased during the selected period.
- Monthly returns were volatile.
- The stock may be attractive for growth-oriented investors, but it also involves risk.

## Important Note
This project uses WRDS data. Reproducing the data extraction step requires institutional access to WRDS.
